# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-07

### Added

- Initial release of Azure Application Gateway v2 module
- WAF_v2 SKU with Web Application Firewall policy
- OWASP 3.2 and Bot Manager managed rule sets
- Autoscaling configuration with min/max capacity
- Multi-site hosting with SNI support
- End-to-end TLS with trusted root certificates
- HTTP-to-HTTPS redirect configurations
- URL rewrite rule sets with conditions
- Custom health probes with match conditions
- Availability zone deployment
- HTTP/2 protocol support
- User-assigned managed identity integration
- Basic and complete usage examples
